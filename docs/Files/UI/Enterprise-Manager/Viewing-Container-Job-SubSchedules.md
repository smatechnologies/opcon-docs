# Viewing Container Jobs' SubSchedules

A Container job runs a SubSchedule — a child schedule nested within the parent schedule. Use this procedure to open and view that SubSchedule directly from the PERT view in Enterprise Manager.

To view a Container job's SubSchedule, complete the following steps:

1. Select **PERT** under the **Operation** topic. The **PERT** screen displays.
2. Select the schedule that contains the Container job.
3. Right-click the Container job box.
4. Select **Open SubSchedule** from the context menu.

**Result:** The PERT view refreshes to display the jobs within the SubSchedule.
