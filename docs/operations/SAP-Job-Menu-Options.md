# SAP Job Menu Options

SAP jobs in Enterprise Manager include two right-click menu options: **SAP Child Processes** and **SAP Job Spools**. These options are available when you right-click an SAP R3/CRM job in an Operations view.

## SAP Child Processes

The **SAP Child Processes** dialog displays the child processes associated with a running or completed SAP job. You can filter the list by status or name, and restart selected child processes when the job has reached a terminated state and you hold the required restart privilege.

The dialog refreshes automatically every 30 seconds.

### Status filter options

The following options filter the child processes displayed in the table. All options are selected by default.

| Option | Description |
|---|---|
| **Released** | Shows child processes with a Released status |
| **Active** | Shows child processes with an Active status |
| **Finished** | Shows child processes with a Finished status |
| **Failed** | Shows child processes with a Failed status |
| **Restarting** | Shows child processes with a Restarting status |

### Table columns

| Column | Description |
|---|---|
| **Process** | Name of the child process |
| **Status** | Current status of the child process |

### Filter field

A **Type Filter Text** input field filters the child process list by partial or full process name. This is a text input, not a button.

### Buttons

| Button | Description |
|---|---|
| **Check All** | Selects all child processes in the list |
| **Uncheck All** | Clears all child process selections |
| **Refresh** | Refreshes the child process list immediately |
| **Restart** | Restarts the selected child processes; enabled only when the job is in a terminated state and you have the Restart Jobs privilege |
| **Close** | Closes the dialog |

## SAP Job Spools

The **SAP Job Spools** dialog lists the spool files generated by an SAP job. Double-click a spool entry to open the **SAP Spool Details** dialog, which displays the spool content.

### Buttons

| Button | Description |
|---|---|
| **Refresh** | Refreshes the spool list |
| **Close** | Closes the dialog |

## SAP Spool Details

The **SAP Spool Details** dialog opens when you double-click a spool entry in the SAP Job Spools dialog. It displays the content of the selected spool file.

### Buttons

| Button | Description |
|---|---|
| **Copy To Clipboard** | Copies the displayed spool content to the clipboard |
| **Open external editor** | Opens the spool file in the system default text editor |
| **Close** | Closes the dialog |
