# Finding Tags

The **Tags** tab in Job Master includes a **Choose Tag** field with a **Find** button. Use the **Find** button to open the **Quick search** dialog and locate a specific tag by name without scrolling through the full list.

To find a tag, complete the following steps:

1. Go to **Administration** and select **Job Master**. The **Job Master** screen displays.
2. Select the schedule in the **Schedule** field.
3. Select the job in the **Job** field.
4. Select the **Tags** tab in the **Job Properties** frame.
5. Select the **Find** button to the right of the **Choose Tag** field. The **Quick search** dialog displays.
6. Type the tag name in the **Select an Item** field.
7. Select **OK**.

The tag matching your search appears selected in the **Choose Tag** field, ready to be added to the job.
