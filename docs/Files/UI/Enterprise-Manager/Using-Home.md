# Using Home

**Home** is the first item under the **Solution Manager** category in the Navigation Panel. Selecting it opens the Solution Manager **Home** page in an embedded browser panel within Enterprise Manager.

:::note
The **Solution Manager** category only appears in the Navigation Panel when a **Solution Manager URL** is defined in the General Server Options. For more information, refer to [Managing Solution Manager](Working-with-Solution-Manager.md).
:::

For a description of the Solution Manager Home page layout, including the Navigation menu, Solutions panel, and Status Bar, refer to [Understanding the User Interface Layout](../Solution-Manager/SM-UI-Layout.md) in the Solution Manager online help.

## Requirements

- The **Solution Manager URL** General Server Option must be configured before the **Solution Manager** category and its **Home** link appear in the Navigation Panel.
- If the Solution Manager URL server option is not configured or is empty, Enterprise Manager displays the message: "Could not access Solution Manager Server Option URL".
