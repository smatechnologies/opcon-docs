# Responsive Platform

Solution Manager is a browser-based interface that adapts its layout to any device or browser window size. Whether you access it from a smartphone, tablet, or desktop, the interface reorganizes itself to fit the available screen width.

## How Responsive Layout Works

Solution Manager uses a responsive design that detects the width of the browser viewport and adjusts the page layout accordingly. On narrow viewports (up to 500 px wide), the interface switches to a mobile layout: the side navigation menu collapses and is replaced by a menu button in the title bar, and page elements reflow to fit the smaller screen.

On wider viewports, the full desktop layout is displayed with the side navigation visible by default.
