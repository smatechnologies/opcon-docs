# Adding Holidays or Non-Working Days to Calendars

**Theme:** Configure  
**Who Is It For?** System Administrator, Automation Engineer

## What Is It?

Use this procedure to add Holidays or Non-Working Days to Calendars in the Enterprise Manager.

To add a holiday or non-working day, complete the following steps:

1. Select **Calendars** under the **Administration** topic. The **Calendars** screen displays
2. Select the **holiday calendar** (either Master Holiday or an individual holiday calendar) in the **Selection** list
3. Select the **days** to mark as holidays or non-working days (displayed in red)
4. Select **x** to the right of the **Calendars** tab to close the screen

## FAQs

**Q: Where do you access Holidays or Non-Working Days to Calendars in the Enterprise Manager?**

Select **Calendars** under the **Administration** topic in the Enterprise Manager navigation pane.

**Q: How do you save a new holidays or non-working days to calendars record?**

After completing the required fields, select the **Save** button on the toolbar to save the holidays or non-working days to calendars record.

## Glossary

**Enterprise Manager (EM)**: OpCon's rich client graphical user interface for Windows and Linux, used to define schedules and jobs, manage automation data, and perform operational tasks.

**Calendar**: A named collection of dates in OpCon used by schedules and frequencies to determine when automation runs or is excluded. Calendars can represent holidays, working days, or any custom date set.
