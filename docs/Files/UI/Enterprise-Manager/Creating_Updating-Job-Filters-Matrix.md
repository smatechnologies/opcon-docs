# Creating/Updating Job Filters

Job filters in Enterprise Manager let you narrow the jobs displayed in the Matrix view by job status, department, access code, or tag. You can create or update a filter at any time to focus on the jobs that matter.

## Create or Update a Job Filter

To create or update a job filter in the Matrix view, complete the following steps:

1. In Enterprise Manager, select **Matrix** under the **Operation** topic. The **Matrix** view opens.
2. On the **Matrix** toolbar, select the **Job Filter** button. The **Job Filter** dialog opens.
3. Select the tab for the criteria you want to filter by: **Job Statuses**, **Departments**, **Access Codes**, or **Tags**.
4. Select one or more items in the list, or select **Select/Unselect All** to include all items on that tab.
5. Repeat steps 3–4 for any additional filter tabs you want to configure.
6. Select **OK** to apply the filter, or select **Cancel** to discard your changes.

**Result:** The Matrix view title updates to **Matrix (Filtered)** when one or more filter criteria are active.

:::note
To clear all active filter criteria at once, open the **Job Filter** dialog and select **Reset**.
:::
