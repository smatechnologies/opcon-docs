# Getting Started

Enterprise Manager is OpCon's desktop graphical user interface for Windows. After logging in, you can customize the application and retrieve installation information using the tasks described below.

## Initial setup tasks

The following tasks are available to help you configure Enterprise Manager after your first login:

- [Managing Connection Profiles](Managing-Connection-Profiles.md) — Create and manage the connection profiles that Enterprise Manager uses to connect to an OpCon database.
- [Setting Preferences](Setting-Preferences.md) — Customize Enterprise Manager's appearance and behavior.
- [Setting the Language](Setting-Language.md) — Override the default display language.
- [Receiving Installation Details](Receiving-Installation-Details.md) — View plug-in names, version numbers, and provider information for the current installation.
