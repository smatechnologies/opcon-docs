# Changing Focus of Days in Calendars

The Matrix view in Enterprise Manager includes a calendar widget with navigation buttons that let you move focus to a specific date. Dates that have schedules built in the daily tables appear with a green background.

To change the focused date in the Matrix view calendar, complete the following steps:

1. In the navigation panel, select **Matrix** under the **Operation** topic.
2. Select one of the following navigation buttons to move focus:

   | Button | Action |
   |--------|--------|
   | **First** | Moves focus to the first available date in the daily tables. |
   | **Today** | Moves focus to the current date, if it is available. |
   | **Last** | Moves focus to the last available date in the daily tables. |

3. Select a date that has a green background to set it as the active date.
4. Select a schedule to view its job information in the daily tables.

**Result:** The Matrix view updates to display the jobs and schedule information for the selected date.
