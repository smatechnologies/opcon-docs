# Getting Support Information

The **Support** dialog in Enterprise Manager displays Continuous contact information and product version details. Review this information before contacting support.

## View Support Information

To view support information, complete the following steps:

1. In the navigation panel, select **Support** under the **Support** category. The **Support** dialog opens.
2. Review the **Contact Information** section for Continuous contact details, including mailing address, websites, email addresses, and phone numbers.
3. Select the **Product Version** tab to view version details for the OpCon release, Enterprise Manager, database schema, stored procedures, database functions, SMALookup, audit triggers, PDSA Framework, and PDSA Data.
4. Select **OK** to close the dialog.

**Result:** The **Support** dialog closes and you return to Enterprise Manager.

:::note
If your user account has the View License Info privilege, a **License Information** tab also appears in the dialog.
:::

## Report a Problem

To report a problem from the Support dialog, complete the following steps:

1. In the navigation panel, select **Support** under the **Support** category. The **Support** dialog opens.
2. Select **Report Problem**. The Report Problem wizard opens.
3. Complete the wizard steps and submit your report.

**Result:** Your problem report is submitted using the SMTP settings configured in OpCon.

## View Installation Details

To view installation details from the Support dialog, complete the following steps:

1. In the navigation panel, select **Support** under the **Support** category. The **Support** dialog opens.
2. Select **Installation Details**. The Installation Details dialog opens and displays component and configuration information for the Enterprise Manager installation.
3. Select **Close** to close the Installation Details dialog.
