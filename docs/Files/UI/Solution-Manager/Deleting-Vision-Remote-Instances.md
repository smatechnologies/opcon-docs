# Deleting Vision Remote Instances

**Theme:** Configure  
**Who Is It For?** System Administrator, Automation Engineer

## What Is It?

Use this procedure to delete Vision Remote Instances in Solution Manager.

To delete a Vision Remote Instance, complete the following steps:

1. Select the **Remote Instances** button on the **Vision Live** or **Vision Settings** page
2. Select the option for the remote instance to delete
3. Select the **Delete** button
4. Select **Yes**
:::

## FAQs

**Q: Can a vision remote instances record be recovered after deletion?**

No. Deleting a vision remote instances record permanently removes it from OpCon. Verify the record is no longer needed before deleting it.

**Q: How many vision remote instances records can you delete at once?**

Select the specific vision remote instances record you want to delete, then select the **Delete** button on the toolbar. Confirm the deletion when prompted.
