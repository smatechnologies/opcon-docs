# Using Operations

The **Operation** section under **Solution Manager** in Enterprise Manager provides links that open the corresponding operational pages in Solution Manager. Each link opens Solution Manager in an embedded browser view and navigates directly to the selected page.

## Operation Links

| Link | Solution Manager Page | Description |
|---|---|---|
| **Summary** | Operations Summary (`operations`) | Displays graphical dials showing high-level status for schedules, jobs, and Agent machines, with drill-down capability. |
| **Processes** | Processes (`operations/processes`) | Displays daily schedule and job activity, allowing you to monitor and act on running work. |
| **Schedule Build** | Schedule Build (`operations/schedulebuild`) | Allows you to build daily schedules for one or more dates. Requires the **Build Daily Schedules** departmental function privilege. |
| **Agent Status** | Agent Status (`operations/agents`) | Displays the status of Agent machines and allows you to perform status updates. |

## Navigation

The **Operation** links are located under **Solution Manager** in the Enterprise Manager navigation pane.
