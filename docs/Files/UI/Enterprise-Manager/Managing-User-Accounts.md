# Managing User Accounts

**Theme:** Configure  
**Who Is It For?** System Administrator, Automation Engineer

The **User Accounts** editor defines and maintains OpCon User Accounts. It displays a list of users, their account details, and configuration options. A user must be a member of a role with All Administrative Functions, All Function Privileges, or the `ocadm` role to manage User Accounts.

For conceptual information on manual login, automatic login, and optional LDAP integration, refer to [User Accounts](../../../administration/user-accounts.md) in the **Concepts** online help.

## User Accounts Toolbar

The **User Accounts** editor toolbar provides buttons for managing user accounts. Select any button to learn more about its functionality.
