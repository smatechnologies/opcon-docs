# Editing Calendars

To edit a calendar:

1. Click the **Edit** button. The **Calendar** dialog displays.
2. Make the desired changes to the calendar. To remove all dates, click **Clear**.
3. Click **Save** to save the updated calendar or **Cancel** to discard changes.
