# Forecasting Frequencies

The **Forecast** feature lets you preview which dates a frequency will include or exclude over a calendar year. Use this to verify frequency rules before applying them to jobs or schedules.

## Forecast a frequency

To forecast a frequency, complete the following steps:

1. Go to **Library** > **Frequencies**.
2. Select the frequency you want to preview.
3. Select **Forecast** in the toolbar.
4. Review the yearly calendar in the **Forecast** dialog. Included dates and excluded dates are highlighted in different colors.
5. Select the left or right arrow button to move to the previous or next year.
6. Select **Close** to close the dialog.

**Result:** The **Forecast** dialog closes and you return to the **Frequencies** list.
