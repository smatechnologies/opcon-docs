# Using Management Tools

The Management tools perform actions on the active [editor](Navigation-Editors.md) or [view](Navigation-Views.md). The available buttons depend on which editor or view is currently open.

## Management Tool Buttons

| Button | Name | Description |
|---|---|---|
| | **Minimize** | Minimizes the editor or view and places an icon for it in the sidebar. |
| | **Maximize** | Opens the editor or view to its maximum size. The button changes to **Restore** when the editor or view is maximized. |
| | **Restore** | Returns the editor or view to its original location and size. |
| | **View Audit** | Opens the **Audit Management** dialog to view the audit record for the selected item. |
| | **Lookup** | Opens the **Notification Lookup** dialog. Enter a Notification ID to look up the source information for that notification. |
| | **Add** (Ctrl+N) | Adds the appropriate record in the active editor. |
| | **Remove** (Ctrl+D) | Deletes the selected record in the active editor. |
| | **Save** (Ctrl+S) | Saves the current changes in the active editor. |
| | **Cancel** (Ctrl+R) | Cancels unsaved changes in the active editor. |
| | **Edit a Frequency Definition** | Opens the frequency wizard to modify the frequency definition rules for the selected frequency. |
| | **Rename** | In Frequency Manager, renames the selected frequency. |
| | **Forecast** | Opens the **Forecast** dialog for the selected frequency. |
| | **Compare** | Compares the frequency definition rules for two selected frequencies side by side. Select exactly two frequencies before using this button. |
| | **Edit Selected Job or Schedule** | Opens the selected job or schedule in its respective editor. |
| | **Associate Additional Jobs** | In Tag Manager, associates a tag with additional jobs or schedules. |
