## What Is It?

Otto is the AI-powered chatbot in Solution Manager for cloud customers. Otto understands natural language questions about OpCon and can perform operational actions using built-in skills — such as restarting failed jobs, checking schedule status, and retrieving application logs.

NOTE: Otto is available exclusively to cloud customers. On-premises customers see a different interface that does not include Otto capabilities.
