# Using Secondary Windows

Enterprise Manager displays secondary windows when you right-click or double-click items in the **Operations** view. Each secondary window gives you details and, in some cases, editing capability for the selected object without opening a separate editor.

The following secondary windows are available depending on the object you select:

| Window | When It Appears |
|---|---|
| [Schedule Information](Schedule-Information.md) | Right-click a schedule in the **List**, **Matrix**, or **PERT** view |
| [Job Information](Job-Information.md) | Double-click a job, or right-click a job and select **Job Information** in the **List**, **Matrix**, **Gantt**, or **PERT** view |
| [SubSchedule Information](SubSchedule-Information.md) | Right-click a subschedule in the **List**, **Matrix**, or **PERT** view and select **SubSchedule Information** |
| [Schedule History](Schedule-History.md) | Right-click a schedule and select **Schedule History** in the **List**, **Matrix**, or **PERT** view |
| [Job History](Job-History.md) | Right-click a job and select **Job History** in the **List**, **Matrix**, or **PERT** view |
| [Machine Information](Machine-Information.md) | Right-click a machine on the **Table** tab or **Map** tab of the **Machine Status** view and select **Machine Information** |
| [Managing SAP R/3 and CRM Background Functions](Managing-SAP-R3-and-CRM-Background-Functions.md) | Right-click an SAP R/3 or CRM machine on the **Machine Status** view and select **Background Functions** |
