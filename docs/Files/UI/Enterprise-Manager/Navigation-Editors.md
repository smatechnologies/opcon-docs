# Navigation Editors

Navigation Editors provide screens to update information stored in the OpCon database. Examples include **Schedule Master**, **Job Master**, **Calendars**, **Global Properties**, **Thresholds**, **Resources**, **Machines**, **Machine Groups**, **Server Options**, **Roles**, **User Accounts**, **Batch Users**, **Departments**, **Access Codes**, and **Privileges**. Any number of editors can be open simultaneously, but only one can be active at a time.

## Opening Navigation Editors

To open a Navigation Editor, complete the following steps:

1. Select the arrow to expand a Navigation topic in the [Navigation Panel](Navigation-Panel.md).
2. Select the **Navigation Editor** (for example, **Schedule Master**).
3. *(Optional)* Drag the **Navigation Editor** tab onto the tab bar if it is not already attached.
4. Repeat step 2 to open additional editors.

**Result:** The selected editor opens in the workspace area and becomes the active editor.

## Changing Navigation Editor Tab Order

To reorder Navigation Editor tabs, drag and release a **Navigation Editor** tab to the new position on the tab bar.

:::note
A stack symbol (![Three white folders with black borders stacked](../../../Resources/Images/EM/stacked-folders.png "Stacked icon")) appears as you drag over other editor tabs.
:::

## Arranging Navigation Editors

To arrange Navigation Editors in the workspace, complete the following steps:

1. Drag the **Navigation Editor** tab toward the position where you want it to display (left, right, top, or bottom).
2. Release when the stack symbol changes to an arrow.

**Result:** The editor drops into place at the selected position.

## Right-Click Menu for Navigation Editors

Right-clicking a **Navigation Editor** tab displays the following options:

| Option | Description |
|---|---|
| **Restore** | Restores the editor to its previous location. |
| **Move** > **Editor** | Moves the individual editor to another location. |
| **Move** > **Tab Group** | Moves the entire grouped set of editors to another location. |
| **Size** > **Left / Right / Top / Bottom** | Highlights the corresponding border edge so you can drag the editor pane to a preferred size. |
| **Minimize** | Minimizes the editor and places it on the right vertical sidebar. |
| **Maximize** | Maximizes the editor to full size. |
| **Close** | Closes the open editor. |
| **Close Others** | Closes all other open editors. |
| **Close All** | Closes all open editors. |
| **New Editor** | Opens a new instance of the currently selected editor. |
